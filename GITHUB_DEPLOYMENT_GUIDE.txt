Issue: App deploys but shows "ModuleNotFoundError"
Solution:
  - Verify requirements.txt has all dependencies
  - Check that .gitignore excludes venv/
  - Streamlit Cloud will auto-install from requirements.txt

Issue: App runs locally but not on Streamlit Cloud
Solution:
  - Make sure .env file is in .gitignore (secrets won't work if .env is committed)
  - Use Streamlit Cloud secrets instead (Settings â†’ Secrets)
  - Check if you're using Windows-specific paths (use Path() from pathlib)

Issue: API keys not working on Streamlit Cloud
Solution:
  - Don't commit .env file (it's in .gitignore)
  - Add keys through Streamlit Cloud Settings â†’ Secrets
  - Wait 30 seconds after adding secrets for app to reload
  - Verify keys are formatted correctly

Issue: Pages showing blank or 404
Solution:
  - Verify main file path is "app_new.py"
  - Check that pages/ folder exists in GitHub repo
  - Ensure all imports are relative (use pathlib for paths)
